GIỚI THIỆU HTML

  • HTML là từ viết tắt của "HyperText Markup Language".
  • HTML là ngôn ngữ đánh dấu tiêu chuẩn cho các trang web.
  • Với HTML, bạn có thể tạo trang web của riêng mình.
  • HTML được coi như là khung xương của một trang web.
  • HTML, CSS, Javascript là bộ 3 không thể thiếu đối với xây dựng phần mềm.
